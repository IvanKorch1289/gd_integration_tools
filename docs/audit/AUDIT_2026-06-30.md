## M. Final Verdict

### Maturity (1–10)
- Architectural: **8/10** (хорошо структурировано, facade pattern, R3.10d compliance).
- Extensibility: **8/10** (PluginLoader, capability-gate, lightweight routes).
- Production readiness: **8/10** (covered by 1510 tests, sandbox defaults now safe).
- DSL completeness: **9/10** (115+ processors, declarative Python+YAML dual mode).
- Agent safety: **7/10** (S3 fix + S1+S2 fixes; остаётся human-engineered policy YAML без typed validator).
- Docs maturity: **8/10** (Diataxis, 16 tutorials, 70 ADR, Vale, Sphinx).
- Maintainability: **7/10** (некоторые god-modules, manual layer violations, docstring debt 649).

### What is Already Good (do NOT break)

- Capability-gated facades (V11.1) — critical for extensions.
- Layer separation enforced через `tools/check_layers.py` AST.
- Extension SDK surface в `src/backend/sdk/__init__.py` (83 LOC).
- DSL dual-mode (Python builder + YAML), 80/20 split.
- YAML hot-reload с SHA-256 caching (P5 fix).
- Sandbox default = ProcessPool (S3 fix), InProcess opt-in с audit.
- Module whitelist в SkillRegistry (S2 fix).
- Tool policy на actual tool name (S168 + S1 fix).
- Symlink escape защита в AIFsFacade.
- file_watch asyncio.to_thread.
- Async-First везде (FastAPI + Temporal).
- WAF strict policy через `core/net/`.
- 3-mixin jupyter execution service.
- Consent to Linter/Typecheck/Test pipeline.

### What Must Be Isolated Before Scaling

1. **Admin endpoints auth** — без `Depends(require_auth)` сейчас, только feature flag.
2. **WebSocket auth** — нет explicit auth на handshake.
3. **Core layer violations** (3 файла в allowlist) — long-term долг.
4. **Argon2id API key** — SHA-256 без соли остаётся HIGH risk.

### Dangerous to Ship to Prod (Today)

- admin_plugins БЕЗ auth (HIGH).
- API key SHA-256 без соли (HIGH, документировано).
- WebSocket БЕЗ explicit auth (MEDIUM-HIGH).

### Stable Public API for Extensions

- `src/backend/sdk/__init__.py` — Pipeline, Exchange, get_service, register_factory, app_state_singleton, BaseError, Clock.
- `src/backend/core/plugin_runtime/manifest.py` — BasePlugin contract.
