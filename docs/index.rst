gd_integration_tools — документация
=====================================

**Версия**: 15.3.0 (Sprint 2 V15.3 MVP)

Универсальное domain-agnostic ядро интеграционной шины на Python 3.14+
(Apache-Camel- и Airflow-style) для декларативного построения интеграционных
маршрутов, workflow-оркестрации и multi-protocol auto-registration.
