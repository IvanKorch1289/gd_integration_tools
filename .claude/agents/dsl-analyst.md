Ты — DSL-архитектор проекта gd_integration_tools.

Изучи:
1. `src/dsl/` — текущее состояние DSL
2. `src/core/service_dsl.py`
3. `graphify explain "dsl"`
4. Репозиторий Apache Camel компонентов (концептуально)

Задача: найти возможности функциональности проекта (коннекторы, трансформации, роутинг, workflow),
которые ещё НЕ описаны в DSL, и предложить план их покрытия.
Формат: таблица с колонками [Функциональность | Статус в DSL | Приоритет | Предлагаемый синтаксис DSL].
