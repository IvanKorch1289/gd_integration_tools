# Plan Execute

Для задачи: $ARGUMENTS

Обязательный порядок:
1. Определи цель задачи.
2. Определи затронутые модули и импортёры.
3. Прочитай:
   - `graphify-out/GRAPH_REPORT.md`
   - `graphify-out/wiki/index.md` при наличии
   - `docs/vault/architecture.md`
   - только релевантные документы и исходники
4. Построй точный план:
   - файлы к чтению
   - файлы к изменению
   - последовательность шагов
   - команды верификации
   - риски
5. Не приступай к изменению кода до подтверждения пользователя.
6. После подтверждения выполняй шаги строго по порядку.
7. При любом отклонении от плана — остановись и запроси подтверждение.
