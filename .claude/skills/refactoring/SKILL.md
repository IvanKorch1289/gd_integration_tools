description: Воркфлоу безопасного рефакторинга с smoke-тестами на каждом шаге
---

## Шаг 1 — Анализ
```bash
graphify explain "<модуль>"
make test    # зафиксировать baseline
```

## Шаг 2 — План (СТОП, жди подтверждения)
- Что меняем (только структура)
- Что НЕ меняем (публичные интерфейсы)
- Риски и зависимые модули

## Шаг 3 — Итеративный рефакторинг
После каждого изменения:
```bash
make test    # smoke-test
make lint
```
При падении тестов — СТОП, сообщить пользователю.

## Шаг 4 — Ревью
Запусти: субагент `code-reviewer`

## Шаг 5 — Финал
```bash
graphify update .
git commit -m "refactor: <описание>"
```
