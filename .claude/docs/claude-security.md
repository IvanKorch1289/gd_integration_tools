# Безопасность Claude-конфигурации

## Что защищено
- `.env`
- `.env.*`
- `secrets/`
- `*.pem`
- `*.key`
- файлы с `secret` и `token` в имени

## Правила
- Не хранить реальные токены в `.mcp.json`
- Передавать секреты только через переменные окружения
- Не ослаблять deny-правила в проектном `.claude/settings.json`
- Любые локальные послабления помещать только в `.claude/settings.local.json`

## Рекомендуемые переменные окружения
- `GITHUB_TOKEN`
- `ANTHROPIC_API_KEY`
- и другие сервисные токены по необходимости
