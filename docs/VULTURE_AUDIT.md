## 6. Reachability Rules (FP filter)

A finding is ACTUALLY DEAD only if ALL 8 hold:

1. Symbol name is unique — no `getattr` / `globals()` callers.
2. No decorator on the symbol (`@app.get`, `@router.post`, `@processor`,
   `@service_dsl`, `@cli.command`, `@server.feature`, `@click.group`,
   `@processor_pool.set`).
3. `git grep -nE '\b<symbol>\b' src/backend/` returns 0 in tests AND
   0 in `__init__.py` re-exports.
4. Not in `infrastructure/database/migrations/versions/` (Alembic rev id).
5. Not a generated `*_pb2_grpc.py` (DO NOT EDIT).
6. Not under `dsl/cli/` (click auto-discovers commands).
7. Not named `__getattr__` (Python lazy-attr shim).
8. Property does not appear in Pydantic `model_config` / `Field`.

Rules 4–7 alone filter ~93% of the 1994 raw findings.
