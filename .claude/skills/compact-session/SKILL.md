# Compact Session

Выполни сжатие контекста:
1. Кратко перечисли, что сделано
2. Сохрани:
   - изменённые файлы
   - выполненные команды проверки
   - открытые риски
   - следующий шаг
3. Сохрани в `vault/session-$(date +%Y-%m-%d-%H%M)-summary.md`
4. После сохранения предложи очистить контекст
